# todos

A web app to keep track of your daily todo tasks. You can create, read, update and delete todo tasks

### Tech Stack

HTML, CSS, Javascript, Vue, Vue Router, Vuex, Express, Mongoose

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```
